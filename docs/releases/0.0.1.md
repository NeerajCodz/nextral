# neuros 0.0.1 (legacy name)

## Release type

Docs-first initialization release.

## Included

- Python package metadata for `pip install neuros`
- Console script entry point: `neuros`
- CLI help and placeholder command surfaces:
  - `about`
  - `memory` (`status`, `add-file`)
  - `tools` (`list`)
  - `files` (`index`)
- Project architecture scaffolding (memory, tools, files, storage, LangChain integration namespaces)
- Documentation index and getting-started guides

## Not included

- Runtime memory pipeline implementation
- Tool execution implementation
- File ingestion implementation
- LangChain runtime integration implementation

## Note

The project has since been rebranded to **Nextral** and reorganized as a runtime-neutral monorepo.
